# how to

```sh
$ git clone git@github.com:RiceCookers/RiceVotes.git
$ cd RiceVotes
$ bundle install
$ bundle exec rails db:migrate
```

## using cloudinary

getting [cloudinary](https://cloudinary.com) api token and key.
and setting env variables.
